📖 Overview

OSPAT (Online Smart Programming Assessment & Interview Platform) is a full-stack SaaS application designed to conduct real-time technical interviews and coding assessments.

The platform enables interviewers and candidates to communicate through video calls, solve coding problems inside a VSCode-powered editor, receive instant code execution feedback, and collaborate in a secure interview environment.

Unlike traditional coding platforms, OSPAT combines:

🎥 Live Video Interviews
💻 Online Code Editor
⚡ Secure Code Execution
💬 Real-Time Chat
📊 Analytics Dashboard
🔐 Authentication & Access Control

into one unified experience.
